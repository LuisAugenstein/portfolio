# Assets Convention

```
assets
├── tech-stack-icons
    └── svg icons if possible, otherwise 24x24 pixel icons.
├── home
│   └── 640x480 project-images with backgroundcolor #D5E8D1. Let some space at the top left for the title.
└── projects
    └── <project-name>
        └── 2:1 ratio images, favorably: 1200-1280 x 600-650 pixel.
```
